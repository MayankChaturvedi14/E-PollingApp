What is E-polling?

E-polling is a web based application where online election procedure can be done.

What are requirements to use E-polling application?

  • You Should have a Legal Name.
  • You should have a Valid Aadhar Card Number.
  • You Should have a Valid Voter ID Card Number.
  • You Should have a Valid Mobile Number.
  • Your Age Should be more than 18 or more than 18.
  • You Should not have any previous criminal records.
